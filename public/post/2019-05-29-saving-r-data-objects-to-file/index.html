Saving R Data Objects - Wed, May 29, 2019

Are you spending too much time reading large data sets, wrangling, then waiting to convert your data frames to matrices or arrays?

Here’s a solution for you: Use R to save one or multiple data objects to file as RDS or RData files. I got this amazing tip from Emil Hvitfeldt and I’ve already saved significant time from this recommendation.
